Quick Sort vs Merge Sort

📌 Deskripsi
Proyek ini dibuat untuk memenuhi tugas mata kuliah Kompleksitas Algoritma. Program ini mengimplementasikan algoritma Quick Sort dan Merge Sort menggunakan bahasa pemrograman Java serta membandingkan waktu eksekusi keduanya pada berbagai ukuran data.

✨Fitur
* Implementasi Quick Sort
* Implementasi Merge Sort
* Generate data acak
* Pengukuran waktu eksekusi menggunakan `System.nanoTime()`
* Perbandingan performa kedua algoritma

⚙️Teknologi
* Java
* JDK 11
* Visual Studio Code

🚀 Cara Menjalankan
Compile program:
```bash
javac SortingComparison.java
```
Jalankan program:
```bash
java SortingComparison
```

📊 Data Pengujian
Program menguji data dengan ukuran:
* 100 data
* 1.000 data
* 5.000 data
* 10.000 data
* 50.000 data
