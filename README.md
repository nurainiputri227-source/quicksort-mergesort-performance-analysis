# 🚀 Quick Sort vs Merge Sort Performance Analysis

📚 Proyek ini dibuat untuk membandingkan performa **Quick Sort** dan **Merge Sort** menggunakan bahasa pemrograman **Java**.

## 🎯 Tujuan
- Mengimplementasikan Quick Sort dan Merge Sort.
- Mengukur waktu eksekusi menggunakan `System.nanoTime()`.
- Membandingkan performa kedua algoritma pada berbagai ukuran data.
- Menganalisis kesesuaian antara teori dan praktik.

## 📊 Data Uji
- 100 data
- 1.000 data
- 5.000 data
- 10.000 data
- 50.000 data

## ⚙️ Teknologi
- ☕ Java
- 💻 Visual Studio Code
- 🔧 JDK 11

## 📈 Hasil
Pengujian menunjukkan bahwa:
- ⚡ Quick Sort cenderung lebih cepat pada data acak.
- 🔄 Merge Sort lebih stabil dengan kompleksitas `O(n log n)` pada semua kondisi.

## ▶️ Menjalankan Program

```bash
javac SortingPerformance.java
java SortingPerformance
```
